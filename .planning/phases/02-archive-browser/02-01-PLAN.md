**Function 1: readAndSortStories**

Purpose: Read all stories from IndexedDB, sort by timestamp descending (newest first), format dates and durations for display.

Signature:
```typescript
export async function readAndSortStories(
  language: 'vi' | 'en' = 'en',
  obs: ObservationSession = createObservationSession()
): Promise<Array<{
  id: number;
  title: string;
  dateFormatted: string;
  durationFormatted: string;
  audioBlob: Blob;
  timestamp: number;
}>>
```

Implementation steps:
1. Use obs.read('stories_request', {})
2. Call getAllStories() from archive.ts with obs observation
3. Sort by timestamp descending: `stories.sort((a, b) => b.timestamp - a.timestamp)`
4. Map each story to display object with:
   - id, title, audioBlob, timestamp (pass through)
   - dateFormatted: Use Intl.DateTimeFormat(language === 'vi' ? 'vi-VN' : 'en-US', { month: 'long', day: 'numeric', year: 'numeric' }).format(new Date(story.timestamp))
   - durationFormatted: formatDuration(story.durationMs) helper function
5. Use obs.observe() for each transformation step
6. Return with obs.return_('sorted_stories', displayArray)

Helper function formatDuration(ms: number): string
- Takes milliseconds
- Returns "MM:SS" format (e.g., "12:34")
- Implementation: const totalSeconds = Math.floor(ms / 1000); const m = Math.floor(totalSeconds / 60); const s = totalSeconds % 60; return `${m}:${String(s).padStart(2, '0')}`;

**Function 2: deleteStory**

Purpose: Remove a story by ID from IndexedDB permanently. No soft-delete.

Signature:
```typescript
export async function deleteStory(
  id: number,
  obs: ObservationSession = createObservationSession()
): Promise<void>
```

Implementation steps:
1. Use obs.read('delete_request', { id })
2. Validate that database is initialized (same pattern as saveStory)
3. Create readwrite transaction on STORE_NAME
4. Call store.delete(id)
5. Observe result: 'story_deleted' or 'story_not_found'
6. Handle errors as StorageError (reuse phase 1 pattern with classifyError)
7. Return with obs.return_('deletion_complete', undefined)

Both functions follow MARK pattern:
- // MARK: comment at top of function with Purpose, Success, Failure
- obs.read() → obs.step() → obs.observe() → obs.return_() flow
- Error handling with classifyError()
- Contract interface definitions

Copy contract style from archive.ts (SaveStoryContract pattern). Export both functions.
