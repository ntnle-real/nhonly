Scene 'list': StoryList component
- Pass: { stories, currentLanguage, onPlayStory, onDeleteStory, isLoading, error }
- onPlayStory(id): set selectedStoryId = id, scene = 'playback'
- onDeleteStory(id): set selectedStoryId = id, scene = 'delete-confirm'

Scene 'playback': Playback component
- Pass: { story: stories.find(s => s.id === selectedStoryId), currentLanguage, onClose, onPlaybackEnded }

- onPlaybackEnded: setTimeout(() => { scene = 'list' }, 2000) — auto-return after 2s

Scene 'delete-confirm': DeleteConfirm component
- Pass: { story: stories.find(s => s.id === selectedStoryId), currentLanguage, onConfirm, onCancel }
- onConfirm: deleteStory(selectedStoryId), remove from stories array, show toast, scene = 'list' after 3s
- onCancel: scene = 'list', selectedStoryId = null

- Layout: flex flex-col min-h-screen
- Main area: flex-1 px-6 py-8 (matching Phase 1 spacing)
- Scene transitions: transition-opacity duration-[450ms] with opacity-100/opacity-0 and pointer-events-none (Phase 1 pattern)

**Reactive language updates:**
- Whenever currentLanguage changes, re-call readAndSortStories(newLanguage) to update date/duration formatting
- Watch currentLanguage store and update date formatting on toggle

**Error handling:**
- If readAndSortStories fails, show error scene with retry button
- If deleteStory fails, show error toast and keep story in list
- If playback fails (handled in Playback component), return to list with error

Create Playback component that displays a story in full-screen with audio playback controls.

**Component signature:**
```svelte
<script lang="ts">
  export let story: {
    id: number;
    title: string;
    dateFormatted: string;
    durationFormatted: string;
    audioBlob: Blob;
    timestamp: number;
  } | null = null;
  export let currentLanguage: 'vi' | 'en' = 'en';
  export let onClose: () => void = () => {};
  export let onPlaybackEnded: () => void = () => {};

  import { onMount, onDestroy } from 'svelte';
  import { translate } from '$lib/i18n';
</script>
```

**State variables:**
- audioElement: HTMLAudioElement | null
- audioURL: string | null (blob URL for playback)
- currentTime: number = 0 (seconds)
- duration: number = 0 (seconds)
- isPlaying: boolean = false
- hasError: boolean = false
- errorMessage: string = ''

**onMount hook:**
1. Create audioURL: URL.createObjectURL(story.audioBlob)
2. Attach event listeners to audio element:
   - on:timeupdate: update currentTime
   - on:ended: call onPlaybackEnded() and wait 2s before returning
   - on:play: set isPlaying = true
   - on:pause: set isPlaying = false
   - on:error: set hasError = true, capture error message
3. Set audio.currentTime = 0 to initialize

**onDestroy hook:**
1. Pause audio playback
2. URL.revokeObjectURL(audioURL) to clean up memory

**Layout (full-screen centered content):**

- Close button (top-right, 32px × 32px, white/50): onclick={onClose}
  - Text: "✕" (translate(currentLanguage, 'playback_close'))

- Main content (centered, max-width 500px):
  - Story title: Display/Lora, 28px, white/90, centered
  - Metadata: Body, 16px, sand/70 — "{dateFormatted} · {durationFormatted}"
  - Spacer: 24px

  - Progress bar (linear):
    - Background: white/10, height 4px, rounded-full
    - Progress fill: teal-light (#2a8fa3), width = (currentTime / duration) * 100%
    - onclick handler: seek to clicked position (audioElement.currentTime = (e.offsetX / barWidth) * duration)

  - Progress label: Body, 16px, white/60, mono font
    - Format: "{formatTime(currentTime)} / {formatTime(duration)}"
    - formatTime helper: return minutes:seconds.padStart(2, '0')

  - Spacer: 16px

  - Play/Pause button (centered, 52px height, teal-light bg):
    - Text: isPlaying ? "⏸" : "▶" (translate pause/play icons)
    - onclick: if isPlaying then audioElement.pause() else audioElement.play()

  - Replay button (below, 52px height, white/8 bg):
    - Text: "↻" (translate(currentLanguage, 'playback_replay'))
    - onclick: audioElement.currentTime = 0; audioElement.play()

**Error handling:**
- If hasError === true, show error scene instead of player:
  - Heading: translate(currentLanguage, 'playback_error_heading')
  - Body: translate(currentLanguage, 'playback_error_body')
  - CTA: translate(currentLanguage, 'playback_error_cta') — onclick={onClose}
  - Style: same as empty state (centered, Lora Display for heading)

**Transitions:**
- Fade in/out with 450ms opacity (Phase 1 pattern)
- Use class:opacity-100/opacity-0 and class:pointer-events-none

All audio element binding:
```svelte
<audio
  bind:this={audioElement}
  on:timeupdate={handleTimeUpdate}
  on:ended={handlePlaybackEnded}
  on:error={handlePlaybackError}
></audio>
```

The src attribute should NOT be set in template — set via JavaScript: audioElement.src = audioURL after URL.createObjectURL()
